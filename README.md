# 🦎 Natural Lizard Cursor Follower

A fun and interactive **HTML5 Canvas** animation where a realistic lizard smoothly follows the user's cursor while maintaining a fixed distance. The animation features a segmented body, walking leg motion, tapered tail, and natural movement using trigonometry and inverse kinematics principles.



## 📸 Preview

Move your mouse around the screen and watch the lizard follow it with smooth, lifelike motion.

## ✨ Features

- 🦎 Smooth lizard movement
- 🎯 Maintains a fixed distance from the cursor
- 🦴 Segmented body with natural bending
- 🚶 Animated walking legs
- 👀 Realistic head with eyes
- 🦎 Tapered tail design
- 📱 Responsive full-screen canvas
- ⚡ Smooth animation using `requestAnimationFrame()`
- 🎨 Pure HTML, CSS, and JavaScript (No libraries)



## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- HTML5 Canvas API


## 🎯 Concepts Used

- HTML5 Canvas Drawing
- Mouse Event Handling
- Trigonometry (`Math.atan2`, `Math.sin`, `Math.cos`)
- Inverse Kinematics (Segment Following)
- Animation Loop
- Responsive Canvas
- Procedural Animation
